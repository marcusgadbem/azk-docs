# PHP / Wordpress / Mysql / phpMyAdmin

### Gerando o Azkfile.js

```sh
$ cd wordpress-folder
$ azk init
```

### Rodando a aplicação

Para iniciar o ambiente de desenvolvimento

```sh
$ azk start -o && azk logs --follow
```
